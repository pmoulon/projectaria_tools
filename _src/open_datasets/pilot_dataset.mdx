# Aria Pilot Dataset

The Aria Pilot Dataset was launched at CVPR in 2022.

It uses a different file structure compared to our other datasets. Our previous wiki in Aria Data Tools contains [documentation for Aria Pilot Dataset](https://facebookresearch.github.io/Aria_data_tools/docs/pilotdata/pilotdata-index/).
